End-to-End Pipeline

1. User holds Control+1 or Control+2
2. HotkeyManager detects keyDown → SpeechCoordinator.didStartRecording
3. PasteManager saves frontmost app
4. AudioRecorder starts M4A recording (16kHz mono)
5. Menu bar → red mic icon
6. User releases key → didStopRecording
7. AudioRecorder stops, returns file URL
8. File size check (< 1KB = silently discard)
9. Menu bar → "..." (Whisper)
10. WhisperService transcribes via OpenAI API
11. Menu bar → sparkles (Claude)
12. ClaudeService enhances text
13. PasteManager re-activates app + Cmd+V
14. Menu bar → idle
15. Temp file cleaned up

How to Build & Run

git clone https://github.com/MINECUMN/SpeechToText.git
cd SpeechToText
open SpeechToText.xcodeproj
# Xcode: Cmd+R

Or via command line:

xcodebuild -project SpeechToText.xcodeproj -scheme SpeechToText -configuration Debug build

Export for Another Mac

  1. Open project in Xcode
  2. Product → Archive
  3. Distribute App → Copy App
  4. Copy SpeechToText.app to target Mac
  5. On first launch: grant Microphone + Accessibility
  6. Open Settings → enter API keys
