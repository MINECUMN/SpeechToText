Step 2: HotkeyManager

What was implemented

Global hotkey detection via CGEventTap with push-to-talk pattern. Control+1 triggers Standard Mode, Control+2 triggers Social Media Mode.

Architecture

ComponentDetails
Event TapCGEvent.tapCreate with cgSessionEventTap
Events Monitored.keyDown, .keyUp, .flagsChanged
Delegate ProtocolHotkeyManagerDelegate — didStartRecording / didStopRecording
Mode EnumSpeechMode.standard, SpeechMode.socialMedia

Permissions

Accessibility permission required. System Preferences → Privacy & Security → Accessibility → enable SpeechToText.

Step 10: Full Integration

What was implemented

SpeechCoordinator wires all components together: Hotkey → Record → Whisper → Claude → Paste. First-run checks for API keys and permissions.

End-to-End Flow

StepComponentAction
1HotkeyManagerDetect Control+1/2 key down
2PasteManagerSave frontmost app
3AudioRecorderStart recording
4HotkeyManagerDetect key release
5WhisperServiceTranscribe audio
6ClaudeServiceEnhance text
7PasteManagerRe-activate app + Cmd+V

Fallback Behavior

ScenarioAction
Claude key missingPaste raw transcription
Claude API errorPaste raw + notification
OpenAI key missingNotification + open Settings
